Title: Appointment Management System
Purpose: This application allows users to manage appointments, customers, and contacts in a business setting.

Author: Ryan Russon
Contact Information: rrusson@wgu.edu
Student Application Version: 1.0
Date: 06/27/2023

IDE: IntelliJ IDEA Community Edition 2021.2
JDK Version: Java SE 17.0.1
JavaFX Version: JavaFX-SDK-17.0.1 (compatible with JDK 17)

Directions for Running the Program:
1. Ensure that you have Java Development Kit (JDK) installed on your system.
2. Download and install IntelliJ IDEA Community Edition.
3. Clone or download the project source code from [https://github.com/rsrusson/Scheduler-03].
4. Open the project in IntelliJ IDEA.
5. Configure the project to use JDK 17 and JavaFX 17.
6. Set up a MySQL database and import the necessary tables.
7. Update the database connection details in the JDBC.java file.
8. Install the MySQL Connector/J driver (version 8.0.25) and add it to the project's dependencies.
9. Build and run the application from IntelliJ IDEA.

Additional Report Description:
As part of the project, an additional report was generated to track and evaluate appointments according to users.
The report lists all appointments according to the user selected and evaluates how many appointments each user has.
This report helps users stay informed and manage their upcoming appointments effectively.
